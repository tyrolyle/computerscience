MAIN MEMORY

WHAT IS RAM?

Also called as RAM (Random Access Memory).

RAM is the most important short-term memory of your device. It reads information that an application or operating system will need in the future and stores it temporarily to access the CPU, graphics card, or other components that may need it. This data remains in RAM and is easily accessible until you close the program or restart the device. The RAM is then cleared and ready to receive new relevant data.

Like the cache on a CPU, RAM makes it so that other components don’t have to fetch data from the device’s slower storage — like a hard drive or even a solid-state drive (SSD) — every time you request a new browser tab

 

HISTORY:

A Historical Journey:

  • Early Days (1940 s-1950 s):
    • Mercury Delay Lines: Used sound waves in tubes of mercury to store data. Slow and bulky.
    • Cathode Ray Tubes (CRTs): Similar to early televisions, they stored data as charged dots on the screen. Limited capacity and slow access times.
  • Magnetic Drum (1948): A rotating cylinder coated with magnetic material, offering faster access than previous methods.
  • Magnetic Core Memory (1950 s-1970 s): Tiny rings of magnetized material stored data. Reliable and faster than drums, but expensive.
  • Integrated Circuits (ICs) and Semiconductor Memory (1960 s-Present): The revolution! Smaller, faster, and cheaper memory chips using transistors became the dominant technology.

Types of Main Memory (Today):

Dynamic Random-Access Memory (DRAM):

  • Most common type in PCs and laptops.
  • Stores data as electrical charges in capacitors, needing periodic refreshing to retain data (hence "dynamic").
  • Offers good balance of cost, performance, and capacity.

Static Random-Access Memory (SRAM):

  • Faster than DRAM but more expensive and consumes more power.
  • Uses transistors to store data, eliminating the need for refresh.
  • Typically used in smaller capacities for CPU cache due to its speed advantage.

Synchronous Dynamic RAM (SDRAM, DDR SDRAM, etc.):

  • An advanced type of DRAM that synchronizes data transfer with the CPU clock for improved performance.
  • DDR (Double Data Rate) variants offer even faster data transfer speeds.
  • The "DDR" followed by a number (e.g., DDR3, DDR4) indicates the specific generation of SDRAM technology.

Emerging Technologies:

  • Magneto- resistive RAM (MRAM): Combines speed of SRAM with the non-volatility of flash memory (discussed in storage). Still under development for mainstream use.
  • Phase-change Memory (PCM): Another promising technology offering fast access times and good endurance.

 

Generations and their Speeds:

The memory’s total speed is referred to as bandwidth and measured in Mega Transfers per second, but traditionally, you’ll see memory marketed with speeds in megahertz (MHz).

Typical DDR4 memory runs between 2,133 MHz and 3,000 MHz, but some can run upwards of 4,866 MHz for the fastest kits available, and DDR5 memory can go anywhere up to 8,000 MHz in some kits. You’ll see these marketed as DDR4-2133 or similar, sometimes with the confusing “PC” label. The number following “PC” is simply the MHz speed multiplied by eight and then rounded. For example, you might see it listed as DDR4-2133 PC4-17000.


Having more than the minimum amount of RAM is essential. A PC runs not only the current application but also other services and tasks in the background. However, having massive amounts of system memory doesn’t necessarily make a PC run faster, and once you have enough, it’s better to have faster memory, than simply more of it.

 

Another common term, especially in the video game space, is VRAM (video RAM). Although once a stand-alone piece of technology, VRAM is currently used to denote dedicated memory on the graphics card. For game consoles, it can also reference system memory, but in either case, it has to do with memory reserved exclusively for the GPU.

 

  • RAM (Random Access Memory): Serves as the main system memory for your computer. It stores data and instructions that the CPU (Central Processing Unit) needs to actively work on. Think of it as your workspace desk, holding the documents and tools you're currently using.
  • VRAM (Video RAM): Dedicated memory specifically designed for graphics processing units (GPU's), also known as graphics cards. It stores data related to graphics processing and rendering, such as textures, frames, and colors. Think of it as a high-speed conveyor belt specifically dedicated to handling visual information for smooth graphical output.

Location:

  • RAM: Located directly on the motherboard of your computer.
  • VRAM: Integrated directly into the graphics card itself.

Speed:

  • RAM: Plays a crucial role in overall system performance, and its speed can impact things like application loading times and multitasking. However, its speeds are typically slower compared to VRAM, ranging from DDR3's 1600 MT/s to DDR5's 5333 MT/s.
  • VRAM: Needs to handle the rapid movement and processing of visual data, so it requires faster speeds than RAM. Typical VRAM speeds range from GDDR6's 16 Gbps (gigabits per second) to the latest GDDR7's 24 Gbps.

Capacity:

  • RAM: Typically comes in capacities ranging from 4 GB to 32 GB for general-purpose computers, with higher capacities becoming more common.
  • VRAM: Generally has smaller capacities compared to RAM, ranging from 2 GB to 16 GB for consumer-grade graphics cards. However, high-end professional graphics cards can have capacities exceeding 24 GB.
